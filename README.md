## КАК ЭТОГО ФРАНКЕНШТЕЙНА ЗАПУСКТЬ?

1. composer update
2. npm install
3. npm run dev || npm run prod
4. переименовать .env.example -> .env
5. php artisan key:generate
6. настроить БД
7. php artisan voyager:install
8. php artisan voyager:admin your@email.com --create
